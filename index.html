ExpenseParser.js 
const extractAmount = (smsString) => {
  // O(N) Regex algorithm
  const regex = /(?:Rs\.?|INR)\s*(\d+(?:\.\d{1,2})?)/i;
  const match = smsString.match(regex);
  if (!match) return null;
  
  return parseFloat(match[1]);
};

Core Innovations

Passive Automation

Background native SMS intercepts drastically reduce manual data entry friction, eliminating human error.

Zero-Trust Security

Rigorous remote JWT validation mitigates XSS, CSRF, and SQLi vectors. Local SQLite fallback for offline routing.

Deterministic Extraction

O(N) Regex algorithms ensure no unhandled computational exceptions during the parsing lifecycle.

Decoupled Architecture

Client UI

React.js (v18+) with purely functional, Hook-driven states and Material-UI styling.

Native Bridge

Capacitor.js (v6.0) for threaded Java injection directly into the JS runtime.

Persistence

Supabase Cloud Infrastructure utilizing PostgreSQL v15 for secure synchronization.
